1.为什么要使用JAVA的BigInteger类？

因为JAVA有很完善的大数运算法则而且在ACM这种比手速的比赛，用C++数组模拟，效率可见一斑。。 因为我懒，懒得用C++数组模拟，Python又不会，只能用java水一下这样子勉强维持生活

2.什么是BigInteger 类？

是JAVA的大整数类运算，可以实现加减乘除等基本运算

3.JAVA BigInteger类 内容的详细介绍

1.从定义开始

（1）常见的函数

BigInteger A  = BigInteger.ZERO
BigInteger B  = BigInteger.ONE
BigInteger C  = BigInteger.TWO
BigInteger D  = BigInteger.TEN

（2）具体解释

1.valueOf(); 强转类型

比如

int a = 3;

BigInteger b = BigInteger.valueOf(a);

b = 3;

再比如

String s = " 12345";

BigInteger c = BigInteger.valueOf(s);

c = 12345

2.add(); 大整数相加

BigInteger a=new BigInteger(“23”);
BigInteger b=new BigInteger(“34”);
a.add(b);

3.subtract()； 相减

4.multiply(); 相乘

5.divide(); 相除取整

6.remainder(); 取余

7.pow(); a.pow(b)=a^b

8.gcd(); 最大公约数

9.abs(); 绝对值

10.negate(); 取反数

11.mod(); a.mod(b)=a%b=a.remainder(b);

12.max(); min();

13.public int comareTo();

返回-1，0或1，分别为BigInteger在数字上小于，等于，或大于值val。

14.boolean equals(); 是否相等

15.BigInteger构造函数：

一般用到以下两种：
BigInteger(String val);
将指定字符串转换为十进制表示形式；
BigInteger(String val,int radix);
将指定基数的 BigInteger 的字符串表示形式转换为 BigInteger

3.例题

1.用JAVA求一个1 ~ n 阶乘和

Code:

package HomeWork1;

import java.math.BigInteger;
import java.util.Scanner;

public class Factorial {
    public static void main(String[] args) {
        Scanner cin = new Scanner(System.in);
        BigInteger res = BigInteger.ZERO;
        int n = cin.nextInt();
        BigInteger temp = BigInteger.ONE;

        for(int i = 1; i <= n; i ++)
        {
            temp = temp.multiply(BigInteger.valueOf(i));
            res = res.add(temp);
        }
        System.out.println(res);
    }
}

2.求一个斐波那契数列

package HomeWork1;

import java.math.BigInteger;
import java.util.Scanner;

public class Fib {
    static int N = 1000010;
    public static void main(String[] args) {
        BigInteger[] fib= new BigInteger[1000010];
        Scanner cin = new Scanner(System.in);
        fib[1] = BigInteger.ONE;
        fib[2] = BigInteger.ONE;

        for(int i = 3 ;i  <= 200; i ++){
            fib[i] = fib[i - 2].add(fib[i - 1]);
        }
        // 求和
        BigInteger res = BigInteger.ZERO;
        int n = cin.nextInt();
        for (int i = 1; i <= n; i++) {
            res = res.add(fib[i]);
        }
        System.out.println("fib前n项的和为 ：" + res);

        //求任意一项
        int m = cin.nextInt();
        System.out.println("fib第m项的和为 : "+ fib[m]);
    }
}
