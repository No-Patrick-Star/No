1. Vanya and Lanterns

题目传送门

1.题意

给你一个n，一个m，还有n个a[i] ,分别代表有n个地方有灯笼, 整条街的长度,每个灯笼所在的位置

2.思路

sort一下 比较相邻两个的最大距离然后除2 再与左端点到左边的距离，和右端点到右边的距离取一个max

3.代码

#include <bits/stdc++.h>

using namespace std;

const int N = 1010;
int a[N];

int main()
{
    int n,m; cin >> n >> m;

    for(int i = 1; i <= n; i ++) cin >> a[i];
    sort(a + 1, a + n + 1);

    int res = 0;
    for(int i =  1; i <= n; i ++) res = max(res, a[i] - a[i - 1]);

    printf("%.9f\n", max(max((double)res / 2 , (double)(m - a[n])) ,(double)a[1]));
}

2. Phoenix and Beauty

题目传送门

1.题意

给你一个n和k，还有n个a[i]；你需要往数组所形成的序列里面添加数，使得长度为k的任意子序列的和相等，但没有要求使用最优方案

2.思路

set存储，我们要把已经存在的数直接扩展n个k; 先用一个set存要输入的数，如果set的长度大于k了，那么一定是不合法的；当长度小于k时，我们随便把一些1~n中不存在set里面的数直接放进入set，直到n == k，然后我们直接输出n次set就OK了

3.Skier

题目传送门

1.题意

给你四个方向分别表示东南西北，初始点为0，然后你根据他给的字符串的方向走，每次走相同的一段距离，如果这条路走过就用时1s，如果没被走过就需要5s，问你总共需要多少秒

2.思路

两个点确定一条线段，mp直接记录两个点，上一个点到现在所在点是否被标记过，被标记过res + 1 ，否则 res + 5；提示：题目可能出现环的情况比如 NESWN 之后后就出现环了 很操蛋

3.代码

#include <bits/stdc++.h>

using namespace std;

#define PII pair<int,int>
#define LL long long
#define pb push_back
#define pqp priority_queue<PII,vector<PII>,greater<PII>>
#define pqi priority_queue<int,vector<int>,greater<int>>
#define PSS pair<string,string>
#define in insert

const int N = 5;

int a[N];
vector<int> e[N];
int dist[N];
int book[N],book2[N];
int dx[]=  {0,-1,0,1,0,0},dy[]= {-1,0,1,0,0,0},dz[] = {0,0,0,0,-1,1};
int t;
int n,k, res;
PII p,q;

map<pair<PII,PII>,int> mp;

int main()
{
    ios_base::sync_with_stdio(0); cin.tie(0);
    int t; cin >> t;
    while(t --)
    {
        string s = "";
        mp.clear();
        cin >> s;
        int x1 = 0, x2 = 0,y1 = 0, y2 = 0;
        int res = 0;
        for(int i = 0; i < s.size(); i ++)
        {
            x2 = x1, y2 = y1;
            if(s[i] == 'N') y2 --;
            if(s[i] == 'S') y2 ++;
            if(s[i] == 'W') x2 --;
            if(s[i] == 'E') x2 ++;
            p = {x1,y1} , q = {x2,y2};
            if(mp[{p,q}] || mp[{q,p}]) res ++; // 这就是可能出现环，要特判的地方qwq
            else res += 5;

            mp[{p , q}] ++;
            x1 = x2 ,y1 = y2;
        }
        cout << res << endl;
    }

    return 0;

}

4.总结

第一次用四个key值的map，很涨见识，感谢大佬

2.思路

反正就是CF的题 你就流氓着做呗，他说啥你别信，也别死磕样例，就走非常人的想法。构造一个前n - 1项全为1的序列。 如果最后一项大于n ，那么 k == n 一定成立，输出这个序列就好了

int a[N];
vector<int> e[N];
int dist[N];
int book[N],book2[N];

{
    ios_base::sync_with_stdio(0); cin.tie(0);
    int n,s; cin >>n >> s;

6.Special Permutation

题目传送门

1.题意

构造一个序列 然后 前一项减后一项的的绝对值在2 - 4 之间

2.解法

我是弱智，先从大到小把 1 ~ n 的奇数输出出来，然后输出一下 4 2 然后从六开始输出偶数就完了，在纸上写一下还是非常有必要的，哎

3.代码

#include <bits/stdc++.h>

using namespace std;
const int N = 1010;
bool st[N];

int main()
{
    int t; cin >> t;
    while(t --)
    {
        int n; cin >> n;
        memset(st,false,sizeof st);

        if(n <= 3) puts("-1");
        else
        {
            for(int i = n ; i >= 1; i --) if( i & 1) cout << i  <<' ';
            cout << 4 << ' ' << 2 << ' ';
            for(int i = 6 ; i <= n; i += 2) cout << i <<' ';

            cout << endl;
        }
    }
}

7. Same Parity Summands

题目传送门

1.题意

给你一个n和k, 你需要用k个数来组合出总和为n的一个数集，并且保证这k个数的奇偶性相同。

2.思路

我们可以直接确定最后一个数，看是否与前面的数就行相同；

3.代码

#include <bits/stdc++.h>

using namespace std;

const int N = 110;
int a[N];

int main()
{
    int t; cin >> t;
    while(t --)
    {
        int n,k;  scanf("%d%d",&n,&k);

        int t1 = n - (k - 1);
        int t2 = n - (k - 1) * 2;
        bool f1 = false, f2 = false;
        if(t1 > 0)
        {
            if(t1 % 2 == 0) f1 = true;
            else
            {
                puts("YES");
                for(int i = 1 ; i < k ; i ++) cout << 1 << ' ';
                cout << t1 << endl;
                continue;
            }
        }
        else
        {
            puts("NO");
            continue;
        }

        if(t2 > 0)
        {
            if(t2 % 2 != 0  && f1)
            {
                puts("NO");
                continue;
            }
            else
            {
                puts("YES");
                for(int i = 1 ; i < k ;i ++) cout << 2 <<' ';
                cout << t2 << endl;
            }
        }
        else
        {
            puts("NO");
            continue;
        }
    }
}

9. Johnny and Contribution

1.题意

给你一个图，和要染色的权重，如果对应的每个点都可以染成给定的颜色那么就输出染色的顺序，否则输出 -1；

2.思路

说是模拟，但是我看了好久的题解qwq，还是太菜了。

对于将要染色的点的权重val，与他相连的各个点的权重一定在[1 ~ val - 1]，并且都存在，我们要染色的一定是 当前与其相连的点的权重不存在且最小的正整数点，我们用set 存下一他周围存在的点的权重，如果s.size() != val - 1 那么一定染色不成功。然后再判断一下set里面的点是否都存在于 [1~ val - 1] 之间，如果不存在，输出 -1。 然后将这个点染色。流下了菜狗的泪水qwq

3.代码

#define int long long

#define y second

#define int long long

    return a[x] < a[y];

    }

    for(int i = 1 ;i <= n; i ++) cin >> a[i], p[i] = i;
